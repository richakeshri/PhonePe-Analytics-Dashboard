📊 PhonePe Transaction Analytics Dashboard

📌 Project Overview
This project analyzes PhonePe Pulse data to understand digital payment trends, transaction performance, user growth, and regional adoption across India.

🎯 Business Objective
- Analyze transaction trends across India
- Track user growth and digital payment adoption
- Identify top-performing states
- Generate actionable business insights

🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel
- Data Modeling

📈 Key KPIs
- Total Transaction Amount: ₹121T+
- Total Transaction Count: 72B+
- Registered Users: 3B+
- YoY Growth %

📊 Dashboard Features
- KPI Cards
- State-wise Geo Map
- Transaction Trend Analysis
- User Growth Analysis
- Top Performing States
- Interactive Slicers

💡 Key Insights
- Identified top-performing states in digital payments.
- Analyzed transaction growth trends across years.
- Compared payment categories and user adoption.
- Generated regional performance insights.

✅ Project Outcome
The dashboard converts raw PhonePe Pulse data into meaningful business insights and supports data-driven decision-making.
